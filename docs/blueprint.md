# **App Name**: StudyQuest

## Core Features:

- Secure User Authentication: Provide login and signup pages with secure authentication flows, gradient backgrounds, floating labels, and smooth input animations.
- Document & Notes Upload: Enable users to upload study materials (syllabi, notes) via a drag-and-drop interface within a dashboard layout.
- AI-powered Quiz Generation: Utilize an AI tool to analyze uploaded documents and dynamically generate custom quizzes for subject comprehension.
- Interactive Study Planner: Allow students to create and manage their weekly study timetables with a calendar layout, subject color tags, and editable entries. This data will be persisted.
- Focus Mode Study Timer: Implement a distraction-free study session mode featuring a large central timer and intuitive start/pause/stop controls, suitable for long study sessions.
- AI-driven Quiz Review & Explanations: Deliver animated quiz results, score breakdowns, and an AI tool to provide clear, expandable explanations for correct and incorrect answers.
- Personalized Performance Dashboard: Display a SaaS-style dashboard with stat cards, charts, and graphs to visualize study hours, readiness scores, and weekly activity. This data will be persisted.

## Style Guidelines:

- The primary color, 'Deep Indigo' (#3F51B5, HSL(231, 48%, 48%)), is chosen for its academic and professional calm, providing a strong anchor for key elements.
- The main background color, a light 'Blue-Gray' (#F8FAFC, HSL(210, 29%, 98%)), offers a clean, non-distracting canvas, aligning with the primary hue family while being highly desaturated and bright.
- An accent 'Soft Purple' (#7C3AED, HSL(263, 79%, 58%)) is used to create smooth gradients for primary buttons and hero sections, complementing the primary indigo.
- Functional accent colors are defined for 'Success' (#10B981), 'Warning' (#F59E0B), and 'Error' (#EF4444) states.
- An 'Optional Dark Mode' (#0F172A) color scheme is provided for user preference or evening study.
- Headlines will utilize a 'Inter' (sans-serif) or 'Poppins' (geometric sans-serif) font at weights 600–700, providing a modern and clean aesthetic.
- Body text will use 'Inter' (sans-serif) or 'Roboto' (grotesque sans-serif) at weights 400–500 for high readability and professionalism, with clear hierarchy across page titles (32–40px), section titles (20–24px), and body text (14–16px). Note: currently only Google Fonts are supported.
- Employ a minimal and clean set of icons that are academic-friendly, enhancing interaction without being distracting, similar to Notion's design philosophy.
- Component borders will feature a radius of 12–16px. Cards will have soft shadows and minimal borders for a premium, clean look.
- The layout will feature consistent spacing and a modular design, adapting fully to various screen sizes with a mobile-first approach where sidebars collapse into a hamburger menu and cards stack neatly.
- All interactions will feature smooth transitions (200–300ms), including hover lift on cards, subtle button ripple/glow effects, loading skeletons, and non-intrusive page transitions to maintain a calm and focused user experience.
- Specific animations include smooth input focus states on forms, progress indicators for uploads, and animated score visuals for quiz results, all designed to be subtle and enhancing.
